Semana 4: Hoja electrónica

← Volver al Inicio

Contenidos

  • Características: Las hojas electrónicas permiten organizar, calcular y analizar datos en forma de tablas. Se trabaja mediante celdas que forman filas y columnas.
  • Funcionalidad: Automatiza cálculos con fórmulas, permite graficar datos, crear bases de datos simples y aplicar filtros, ideal para informes y proyecciones.
  • Entorno de trabajo: Compuesto por la barra de fórmulas, celdas, filas, columnas, hojas y menús. Herramientas como Excel y Google Sheets son ejemplos comunes.
  • Funciones y fórmulas: Incluyen operaciones básicas como SUMA(), PROMEDIO(), SI(), BUSCARV(), y más, para facilitar el análisis numérico y lógico.
  • Gráficos: Representan visualmente los datos con diferentes tipos como columnas, líneas, circulares, barras o áreas para una mejor interpretación.
