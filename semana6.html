Semana 6: Servicios de Internet

← Volver al Inicio

Contenidos

  • Web: Es el servicio más popular de Internet, permite acceder a páginas mediante navegadores como Chrome o Firefox. Utiliza protocolos como HTTP y HTTPS para mostrar texto, imágenes, videos y más.
  • Email: Es el sistema de correo electrónico que permite enviar y recibir mensajes digitales a través de Internet. Utiliza protocolos como POP3, IMAP y SMTP. Ejemplos: Gmail, Outlook.
  • FTP: File Transfer Protocol es un servicio que permite la transferencia de archivos entre computadoras conectadas a Internet. Se usa para subir y descargar archivos desde servidores web.
