Semana 1: Hardware

← Volver al Inicio

Contenidos

  • Conceptos: El hardware es el conjunto de componentes físicos y tangibles que conforman un sistema informático. Incluye dispositivos internos como la CPU y externos como el teclado.
  • Rendimiento: Se refiere a la velocidad y eficiencia con la que los componentes del hardware ejecutan tareas. Factores como el procesador, la memoria RAM y el tipo de almacenamiento influyen directamente.
  • Dispositivos E/S: Los dispositivos de entrada/salida permiten la interacción del usuario con la computadora. Entrada: teclado, mouse. Salida: monitor, impresora. Algunos son mixtos, como pantallas táctiles.
  • Memoria y Almacenamiento: La memoria RAM es temporal y volátil; el almacenamiento (como discos duros o SSD) es permanente y guarda datos incluso apagado el sistema.
