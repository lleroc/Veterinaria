Contenidos

  • Conceptos: El software es el conjunto de instrucciones, programas y datos que permiten operar el hardware. Es intangible y esencial para ejecutar tareas en una computadora.
  • Sistemas Operativos: Son plataformas que gestionan los recursos del hardware y permiten la interacción entre el usuario y la máquina. Ejemplos: Windows, Linux, macOS.
  • Tipos de aplicaciones: Existen aplicaciones de sistema (como antivirus), de usuario (navegadores, procesadores de texto) y especializadas (software veterinario, diseño gráfico, etc.).
